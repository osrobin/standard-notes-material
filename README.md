# Standard Notes Dark Material Theme

A clean **Material-inspired dark theme** for **Standard Notes** that gives your editor a modern, easy-on-the-eyes look.

This theme works on mobile phone and desktop.

---

## Install

**Desktop / Web app**

1. Open **Standard Notes**.
2. Go to **Plugins**.
4. Paste this URL under Install Custom Plugin and click install: https://osrobin.github.io/standard-notes-material/ext.json
