# Standard Notes Material Theme

A dark, Material-inspired theme for Standard Notes that gives the UI a modern, easy-on-the-eyes look.

This theme works on both desktop and mobile versions of Standard Notes. ([GitHub][1])

## Installation

1. Open Standard Notes (desktop or web app).
2. Go to **Plugins**.
3. Under **Install Custom Plugin**, paste this URL and press Enter:

```
https://osrobin.github.io/standard-notes-material/ext.json
```

4. Click **Install** when prompted.
5. Activate the theme from the **Extensions** or **Appearance** section. ([GitHub][1])

## About

This is a simple CSS theme that overrides Standard Notes defaults with Material UI inspired colors and styles. It’s fully open source and works wherever Standard Notes supports custom themes. ([GitHub][1])

## Included Files

Here’s what’s in this project:

```
README.md           - This file  
ext.json            - Standard Notes extension descriptor  
package.json        - Project metadata  
theme.css           - The actual theme CSS  
```

* **theme.css**: Core theme styles you can tweak.
* **ext.json**: Tells Standard Notes how to install the theme.
* **package.json**: Basic info for development or packaging.

## Customization

If you want to tweak the theme:

1. Clone this repo.
2. Edit `theme.css` to change colors, fonts, spacing, etc.
3. Build a new extension by updating `ext.json` with your new raw file URLs.
4. Install it through Standard Notes using your URL.
